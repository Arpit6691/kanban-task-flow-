# 🗂️ Mini Kanban Board (Next.js)

A Trello-like Kanban board built using Next.js where users can create, manage, and move tasks across different workflow stages.

---

## 🚀 Features

### 📌 Task Management
- Create new cards with:
  - Title
  - Description
- New cards are added to the **Pending** column by default

### 📊 Board View
- Three columns:
  - 🕒 Pending  
  - ⚙️ In Progress  
  - ✅ Completed  
- Each column displays its respective tasks

### 🔄 Move Cards
- Move tasks between stages:
  - Pending → In Progress → Completed

### ✏️ Edit Cards
- Update task title and description

### 🗑️ Delete Cards
- Delete tasks with confirmation
